# Wonderland XR

An immersive web and VR experience about anxiety, built with A-Frame and Three.js.

The user falls through a rabbit hole into a distorted Wonderland room, where objects inspired by _Alice in Wonderland_ each represent a kind of anxiety. Interacting with them doesn't remove the anxiety — it names it and makes it familiar.

## Run locally

```bash
npm install
npm run dev
```

Then open the local address Vite gives you.
